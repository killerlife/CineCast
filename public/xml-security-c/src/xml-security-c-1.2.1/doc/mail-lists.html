XML security mailing list

There exists a mailing list which you can subscribe to ask questions:

The ezmlm mailing list controller accepts commands by sending emails to it, generally like the following:

  • security-dev-subscribe to subscribe your current email address to the list.
  • security-dev-unsubscribe to unsubscribe your *current* email address from the list.
  • security-dev-help to get Help on mailing list commands.
