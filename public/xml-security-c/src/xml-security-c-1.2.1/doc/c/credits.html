Software Used

The XML-Security-C package makes heavy use of the Xalan-C and Xerces-C++ libraries from the Apache Software Foundation XML Project.

This product includes cryptographic software written by Eric Young (eay@cryptsoft.com). This product includes software written by Tim Hudson (tjh@cryptsoft.com).

This product includes software developed by the OpenSSL Project for use in the OpenSSL Toolkit ( www.openssl.org)

The license documents for Xerces, Xalan and OpenSSL are contained in the xml-security/c/doc directory in CVS and in the doc directory in distributions of the C++ source code.

This documentation has been created using Forrest, also part of the Apache Sofware Foundation's XML project.

API Documentation is created using Doxygen and GraphViz
